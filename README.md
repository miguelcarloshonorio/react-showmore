> Simple ShowMore component

## Install

NPM:
```sh
npm i @micajeho/react-showmore
```

YARN:
```sh
yarn add @micajeho/react-showmore
```

## Usage

```javascript
    <ShowMore 
      minLenth={15}
      content={content} 
      moreText={' More...'} 
      lessText={' ... Less'} 
     />
```
